# 변경 기록

## 0.5.1 - 2026-07-17

- 전체 화면 캡처에서도 상대 선택 카드 6개를 찾아 포켓몬 영역만 안정적으로 비교하도록 인식 범위를 개선했습니다.
- 이전 형식의 학습 이미지를 자동 판정에서 제외하고, 현재 형식의 고신뢰 학습 결과만 자동 입력하도록 보수적으로 조정했습니다.
- 상대 팀 미리보기에 한글·영문 이름 검색과 결과 수 표시를 추가하고 선택 중인 슬롯값을 유지합니다.
- 현재 턴 입력을 내 포켓몬·상대 포켓몬·필드 효과·공개 기술 박스로 재배치하고 기본 창 폭의 반응형 정렬을 개선했습니다.

## 0.5.0 - 2026-07-17

- M-B 235종/폼의 로컬 선택 화면 이미지 참조팩과 자동 갱신을 추가했습니다.
- 상대 선택 화면의 오른쪽 6슬롯을 분리해 로컬 Top-3 후보와 타입 근거를 표시합니다.
- 로컬 후보를 NVIDIA NIM 판독과 결합하고, 사용자가 확정한 슬롯 이미지만 로컬 학습본으로 저장합니다.
- 첨부 샘플의 정답 6종을 확정 학습한 뒤 6/6 모두 로컬 1순위로 재인식되는 흐름을 검증했습니다.

## 0.4.1 - 2026-07-16

- 앱 창의 항상 위 표시를 제거해 다른 창을 선택하면 자연스럽게 뒤로 이동하도록 수정했습니다.
- `내 팀`과 `팀 제작`을 분리하고, 저장한 팀을 불러와 같은 정보로 다시 수정할 수 있게 했습니다.
- 저장한 팀의 포켓몬 조합과 운용 방향을 보여 주는 간단한 팀 전략 요약을 추가했습니다.
- 화면 인식 프롬프트를 팀 선택 화면의 좌우 배치와 상대 6슬롯 기준으로 보강했습니다.
- 선출 추천을 `1순위`, `2순위`, `3순위`로 명확히 정리하고 각 추천의 근거를 표시합니다.
- 잠듦·하품·트릭룸 등 비공격 상태를 수동으로 반영하고, 연속 화면 분석으로 추천을 갱신할 수 있게 했습니다.

## 0.4.0 - 2026-07-16

- 현재 M-B 규정 포켓몬·기술·특성·도구·성격의 한국어 표시와 검색·정렬을 추가했습니다.
- 확인된 사용률을 선택지 정렬에 반영하고, 기본 종족값과 66 능력 포인트 기반 Lv.50 스탯 계산을 추가했습니다.
- 같은 지닌 도구의 중복을 차단하고 대짱이의 웨이브태클을 Champions 확인 기술로 반영했습니다.
- NVIDIA 화면 인식을 상대 팀 6슬롯 후보·신뢰도·시각 근거 구조로 확장했습니다.
- 비·쾌청·모래바람·설경·트릭룸 팀 구조를 선출 추천의 근거와 위험 평가에 반영했습니다.

## 0.3.3 - 2026-07-16

- sandbox renderer에서 지원되지 않는 ESM preload를 CommonJS preload로 전환해 빈 화면 문제를 수정했습니다.
- preload 연결이 실패해도 빈 화면 대신 복구 안내를 표시합니다.
- 릴리스 검증에 sandbox preload 형식 회귀 검사를 추가했습니다.

## 0.3.2 - 2026-07-16

- 기본 업데이트 피드를 `jinjerry0927/pochamp-ai` GitHub Releases에 연결했습니다.
- 기존에 업데이트 주소가 비어 있던 설정을 새 기본 주소로 자동 마이그레이션합니다.

## 0.3.1 - 2026-07-16

- 패키징 환경에서 `electron-updater` CommonJS 모듈을 불러오지 못해 앱이 시작되지 않던 문제를 수정했습니다.
- GitHub Actions 검증 및 Windows Release 초안을 추가했습니다.

## 0.3.0 - 2026-07-16

- Electron sandbox, CSP, 권한 차단, IPC 입력 검증을 적용했습니다.
- 취약한 서버 의존성을 제거하고 `@pkmn/dex` 데이터 패키지로 교체했습니다.
- HTTPS 피드 기반 앱 내 업데이트 확인·다운로드·재시작 설치 기능을 추가했습니다.
